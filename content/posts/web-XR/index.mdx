Anteriormente para renderizar animaciones 2d y 3d en un navegador web el usuario tenía que instalar Plugins que incorporaran esa lógica gráfica dentro de la web.

Entonces en aquellos tiempos se usaba el tan conocido Adobe Flash Player que era un plugin que agrega toda esta lógica pero no venía de forma nativa, por lo cual para que las personas pudieran  visualizar o tener herramientas un poco más potentes en la web se instalaba de manera independiente, actualmente se usa WEBGL el cual es un API gráfica creada por el grupo  [Khronos,](https://en.wikipedia.org/wiki/Khronos_Group) y viene instalada de forma nativa en la mayoría de navegadores modernos.

![Se muestra la pagina de Adobe imagen tomada el 7 nov de 2022](https://i.imgur.com/oeV3yE8.png)

Esto a dado soluciones mas amenas al desarrollador para crear nuevas experciencias 2D, 3D, realidad aumtadas y realidad virtual.
